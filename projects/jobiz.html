Jobiz Application

The aim of this project was to provide an educational application in the form of a quiz and to check your knowledge of technologies used at work as a frontend or backend developer.

Technologies used in project:

  • Miro, for the implementation of the project concept
  • Figma, for the mockup and design refinement
  • GitHub, for parallel code work and project management
  • HTML to define the structure of the application
  • CSS, for styling the application (BEM methodology adopted)
  • JavaScript, to execute the application logic and interactive elements of the interface
  • Web API for JavaScript
  • Fetch API to connect to WEB API
  • LocalStorage, for storing game settings and highscores
  • Snowpack, for building applications and managing dependencies
  • Prettier, for automatic code formatting
  • Github pages and a script to build the project and deploy it with the use of appropriate libraries
